Simple Installation & Usage Guide

This file explains how to install and run the Quiz Project step-by-step.

------------------------------------------------------------------------

STEP 1: Run the Backend (API)

1.  Unzip the project folder.
2.  Open the .csproj file in Visual Studio 2022.
    -   Project uses .NET 9.0
3.  In Solution Explorer:
    -   Right-click the project
    -   Click Set as Startup Project.
4.  Click Run ▶.

If Swagger does NOT open automatically, open your browser and paste this
link: https://localhost:5001/swagger/index.html

IMPORTANT: The API must keep running in the background.

------------------------------------------------------------------------

STEP 2: Run the Frontend (UI)

1.  Open VS Code.
2.  Open the Frontend folder of the project.
3.  Install the extension: Live Server.
4.  Right-click on index.html.
5.  Click Open with Live Server.

Your quiz interface will open in the browser.

------------------------------------------------------------------------

Simple Project Info

This is a TV-game quiz web where: - Users answer
multiple-choice questions - Score is calculated automatically - Result
is shown at the end

It shows the connection between frontend and backend using API.
